
# Follow Projects, Headers, Sources and Objects

Included are four directives:
 - fwp: Follow Projects
	- .fwp:		Global definitions (physics + constants)
 - fwb: Follow Bodies
	- .fwb:		Package graph
 - fwh: Follow Headers
	- .fwh:		Type system
 - fws: Follow Sources
	- .fws:		Executable symbolic source
 - fwo: Follow Objects
	- .fwo:		Compiled unified object
